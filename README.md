# berlin-christmas-markets-2023

In this repo I collect data from the website [https://www.deutsche-weihnachtsmaerkte.de/weihnachtsmarkt/](https://www.deutsche-weihnachtsmaerkte.de/weihnachtsmarkt/) to find all available christmas markets on a daily basis for each single date from 16.11.2023 until 31.12.2023.

Please check the [calendar](src/calendar.md) for each date.
